HyperText Markup Language

HTML is the standard markup language for creating web pages.

It describes the structure of a web page using markup.

  • HTML Element: An HTML element is a building block of an HTML page.
  • Tags: HTML uses tags to define elements. Tags are enclosed in angle brackets.
  • Attributes: HTML elements can have attributes that provide additional information.
  • Document Structure: An HTML document has a structure that includes the head and body sections.
  • Doctype Declaration: The declaration defines the document type and version of HTML.

Sample HTML Code:

<html> <head> <title>My Web Page</title> </head> <body> <h1>Hello, World!</h1> </body> </html>
Cascading Style Sheets

CSS is a style sheet language used for describing the look and formatting of a document written in HTML.

It controls the layout of multiple web pages all at once.

  • Selectors: CSS selectors define the elements to which a style rule applies.
  • Properties: CSS properties determine the visual style of an element.
  • Values: CSS values are assigned to properties to define specific styles.
  • Box Model: The CSS box model describes the layout of elements in terms of content, padding, border, and margin.
  • Cascading: CSS rules cascade, meaning that the styles of multiple rules are combined to determine the final style of an element.
Java Script

JavaScript is a high-level, interpreted programming language.

It is widely used to make web pages interactive and dynamic.

  • Variables: Variables are used to store data values.
  • Functions: Functions are blocks of reusable code.
  • Control Flow: Control flow statements determine the order in which statements are executed.
  • Objects: Objects are used to group related data and functions.
  • Events: JavaScript can respond to user actions or events on a web page.
React Js

React Js is a JavaScript library for building user interfaces.

It allows developers to create reusable UI components.

  • Components: React applications are built using components.
  • State: State is used to manage data that changes over time within a component.
  • Props: Props allow data to be passed from a parent component to a child component.
  • Virtual DOM: React uses a virtual DOM to optimize the updating of the actual DOM.
  • React Router: React Router enables navigation between different components in a React application.
Media Queries

Media queries are a feature of CSS that enable the styling of a web page based on various

characteristics of the device on which it is being viewed.

  • Responsive Design: Media queries are commonly used to create responsive designs that adapt to different screen sizes.
  • Breakpoints: Breakpoints are specific points at which the layout of a website will change in response to different screen sizes.
  • Viewport: The viewport meta tag controls the width and scaling of the viewport on mobile devices.
  • Orientation: Media queries can be used to apply styles based on the orientation of the device (portrait or landscape).
  • Print Styles: Media queries can also be used to define styles specifically for printed documents.
